# utoniq テキストトーン表

> 機能カテゴリ別のテキスト方針。新規機能追加時もこの表を参照してトーンを統一する。
> 具体的なペルソナ情報は `persona.md` を参照。

---

## 凡例

| 記号 | 意味 |
|---|---|
| 🎤 | アーティスト発言（メンバー個別の話し方に従う） |
| 👤 | ファン発言（ペルソナに従う） |
| 📝 | システム / UIテキスト |
| 🏷️ | ラベル・タイトル類 |

## 機能カテゴリ別トーン

### グローバルホーム / GlobalHome
| 要素 | トーン | 例 |
|---|---|---|
| 🏷️ セクション見出し | シンプル・機能的 | 「最新のポスト」「おすすめイベント」 |
| 📝 レコメンド文 | 親しみやすく誘導 | 「ハルトが新しい写真を投稿しました」 |
| 🏷️ バナー | 訴求力重視・短文 | 「voice 発売記念ガチャ開催中」 |

### トップ / Top（アーティスト個別トップ）
| 要素 | トーン | 例 |
|---|---|---|
| 🎤 アーティスト挨拶 | 温かみ・距離の近さ | 「ルミナのみんな、今日も来てくれてありがとう🌙」 |
| 🏷️ メニューラベル | 簡潔 | 「ポスト」「ガチャ」「イベント」 |
| 📝 最新情報 | ニュース調 | 「1stアルバム『voice』好評発売中」 |

### プロフィール / Profile
| 要素 | トーン | 例 |
|---|---|---|
| 🏷️ バイオ | 公式・簡潔 | 「Lumière メインボーカル / 作詞担当」 |
| 📝 自己紹介文 | 本人らしさ | 「歌で届けたい想いがあります。よろしくね🌙」（ハルトの場合） |
| 📝 数値ラベル | 機能的 | 「フォロワー 120,000」「投稿 342」 |

### ポスト / Post
| 要素 | トーン | 例 |
|---|---|---|
| 🎤 投稿本文 | メンバー個別の話し方 | ハルト：「今日のリハで新曲初披露したよ🎵」 |
| 👤 コメント | ファンペルソナに従う | 夏希：「イントロから鳥肌だった…！」 |
| 📝 いいね数 | 数値のみ | 「1.2K」「324」「89」 |
| 🏷️ カテゴリタグ | 短いラベル | 「#リハーサル」「#新曲」「#日常」 |

### ガチャ / Gachas
| 要素 | トーン | 例 |

| 📝 レアリティ | 英字固定 | SSR / SR / R / N |
| 📝 確率表示 | 事実のみ | 「SSR 3%」「SR 12%」「R 35%」「N 50%」 |

## 数値の基準ルール

| 数値 | 基準 | 根拠 |
|---|---|---|
| フォロワー数 | Lumière全体: 32万 / メンバー個別: 6〜12万 | persona.md に準拠 |
| いいね数 | 人気投稿: 1.2K〜3K / 通常: 200〜800 / 少なめ: 30〜100 | グループ規模から逆算 |
| コメント数 | 人気: 150〜400 / 通常: 30〜80 / 少なめ: 5〜15 | いいね数の約10〜15% |
| ガチャ確率 | SSR 3% / SR 12% / R 35% / N 50% | 業界慣行 |
| コイン単価 | 1コイン ≒ 1円 | わかりやすさ重視 |
| ライブ視聴者 | 配信中: 1,000〜3,000 / アーカイブ再生: 5,000〜15,000 | グループ規模から逆算 |
| ランキング | 表示は10位まで。Lumièreは3〜5位程度に配置 | 1位にすると不自然 |

## テキスト文字数の目安

| 箇所 | 目安 | 理由 |
|---|---|---|
| ガチャ景品名 | 全角16文字以内 | カード2行以内に収める |
| ミッション名 | 全角20文字以内 | リスト1行に収める |
| ポスト本文 | 2〜4行 | 1行は間延び、5行以上はスクロール |
| コメント | 1〜2行 | リスト表示の密度 |
| イベント説明 | 2〜3行 | カード内に収まる量 |
| 商品名 | 全角25文字以内 | カード幅に収める |
| 通知テキスト | 全角30文字以内 | 1行表示を想定 |
| ボタンラベル | 全角8文字以内 | ボタン幅に収める |
