# プロフィール画像 URL マッピング

> AI がサンプルテキスト生成時に画像も指定できるよう、各キャラクターに固定のプロフィール画像 URL を割り当てる。
> 画像は `i.pravatar.cc`（アバター画像サービス）を使用。番号固定のため、常に同じ顔が表示される。

---

## 使い方

HTML や Figma で画像が必要な場合、以下の URL をそのまま使用する：
- 正方形: `https://i.pravatar.cc/150?img={番号}`
- サイズ変更: `/150` の数値を変える（例: `/300` で 300×300px）
